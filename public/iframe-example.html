🎯 Maki Aranma Talep Formu - iFrame Entegrasyonu

📋 Kullanım Yöntemleri

Yöntem 1: Kullanıcı Adı ile iFrame (Önerilen - Otomatik Doldurma)

Kullanıcı adını URL parametresi olarak gönderin:

<iframe 
    src="http://localhost:3000?username=kullaniciadi123" 
    width="100%" 
    height="700" 
    frameborder="0"
    style="border: none; border-radius: 8px;"
></iframe>

Not: kullaniciadi123 yerine gerçek kullanıcı adını koyun.

Yöntem 2: Basit iFrame Kodu (Kullanıcı Manuel Girer)

Kullanıcı adı otomatik gönderilmezse kullanıcı kendisi girer:

<iframe 
    src="http://localhost:3000" 
    width="100%" 
    height="700" 
    frameborder="0"
    style="border: none; border-radius: 8px;"
></iframe>

Yöntem 3: Responsive iFrame (Mobil Uyumlu)

<div style="position: relative; padding-bottom: 100%; height: 0; overflow: hidden; max-width: 500px; margin: 0 auto;">
    <iframe 
        src="http://localhost:3000?username=kullaniciadi123" 
        style="position: absolute; top: 0; left: 0; width: 100%; height: 100%; border: none; border-radius: 8px;"
    ></iframe>
</div>

Yöntem 4: JavaScript ile Dinamik Yükleme + Otomatik Kullanıcı Adı

Sitenizin oturum sisteminden kullanıcı adını otomatik çekin:

<div id="makiForm"></div>

<script>
    // Kendi sitenizin kullanıcı adını buradan alın
    const currentUsername = 'john_doe'; // localStorage, cookie veya API'den çekin
    
    const iframe = document.createElement('iframe');
    iframe.src = `http://localhost:3000?username=${encodeURIComponent(currentUsername)}`;
    iframe.style.width = '100%';
    iframe.style.height = '700px';
    iframe.style.border = 'none';
    iframe.style.borderRadius = '8px';
    document.getElementById('makiForm').appendChild(iframe);
</script>

Yöntem 5: PHP ile Dinamik Username (Sunucu Taraflı)

<?php
    // Oturum açmış kullanıcının adını al
    $username = $_SESSION['username']; // veya veritabanından
?>

<iframe 
    src="http://localhost:3000?username=<?php echo urlencode($username); ?>" 
    width="100%" 
    height="700" 
    frameborder="0"
    style="border: none; border-radius: 8px;"
></iframe>

⚙️ Önemli Notlar:

  • http://localhost:3000 adresini canlı domain adresinizle değiştirin
  • HTTPS kullanıyorsanız, iframe src'de de HTTPS kullanın
  • Kullanıcı adı parametresi: ?username=KULLANICIADI şeklinde ekleyin
  • Türkçe karakter: encodeURIComponent() veya urlencode() kullanın
  • Height değerini içeriğe göre ayarlayabilirsiniz (600-800px arası önerilir)
  • İçerik tam görünmüyorsa height değerini artırın
  • Güvenlik: Kullanıcı adını URL'den alırken sanitize edin

🔧 Kullanıcı Adı Kaynakları:

  • JavaScript: localStorage.getItem('username')
  • Cookie: document.cookie veya kütüphane kullanın
  • Session Storage: sessionStorage.getItem('username')
  • API: Backend'den AJAX/Fetch ile çekin
  • PHP: $_SESSION['username']
  • WordPress: wp_get_current_user()->user_login

🎨 Styling İpuçları:

  • Şeffaf arka plan: Form zaten koyu temalı, kendi sitenizin arka planıyla uyum sağlar
  • Border radius: border-radius: 8px ile modern görünüm
  • Shadow: box-shadow: 0 4px 16px rgba(0,0,0,0.1) ile derinlik kazandırın
  • Max width: max-width: 500px ile dar ekranlarda daha iyi görünüm
