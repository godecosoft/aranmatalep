🚀 Gelişmiş iFrame Entegrasyonu - postMessage API

🎯 postMessage API Neden Daha İyi?

✅ Avantajlar:
  • URL'de parametre adı endişesi yok
  • Güvenli (origin kontrolü yapılabilir)
  • Dinamik - istediğiniz zaman veri gönderebilirsiniz
  • Birden fazla veri gönderebilirsiniz (username, email, phone, etc.)
  • URL'de hassas bilgi görünmez
  • İki yönlü iletişim mümkün

📋 Kullanım Yöntemleri

Yöntem 1: postMessage ile Basit Kullanım

Parent sayfanızda bu kodu kullanın:

<!-- HTML -->
<iframe id="makiIframe" src="http://localhost:3000"></iframe>

<script>
// iframe yüklendikten sonra
const iframe = document.getElementById('makiIframe');

iframe.addEventListener('load', () => {
    // Kullanıcı adını iframe'e gönder
    const username = 'john_doe'; // Kendi sisteminizdeki kullanıcı
    
    iframe.contentWindow.postMessage({
        type: 'setUsername',
        username: username
    }, '*'); // Production'da '*' yerine gerçek domain kullanın
});
</script>

Yöntem 2: Dinamik Kullanıcı Bilgisi Gönderme

<script>
// Kullanıcı giriş yaptıktan sonra
function onUserLogin(user) {
    const iframe = document.getElementById('makiIframe');
    
    if (iframe && iframe.contentWindow) {
        iframe.contentWindow.postMessage({
            type: 'setUsername',
            username: user.username // veya user.login, user.name vs.
        }, '*');
    }
}

// iframe'in hazır olduğunu dinle
window.addEventListener('message', (event) => {
    if (event.data && event.data.type === 'iframeReady') {
        console.log('iframe hazır, kullanıcı bilgisi gönderilebilir');
        onUserLogin({ username: getCurrentUser() });
    }
});
</script>

Yöntem 3: URL Parametresi ile (Alternatif - Her Parametre İsmi Desteklenir)

Artık farklı parametre isimleri otomatik tanınır:

<!-- Bunların hepsi çalışır -->
<iframe src="http://localhost:3000?username=john"></iframe>
<iframe src="http://localhost:3000?user=john"></iframe>
<iframe src="http://localhost:3000?userName=john"></iframe>
<iframe src="http://localhost:3000?user_name=john"></iframe>
<iframe src="http://localhost:3000?login=john"></iframe>
<iframe src="http://localhost:3000?account=john"></iframe>
<iframe src="http://localhost:3000?userid=john"></iframe>
⚠️ Desteklenen Parametre İsimleri:
username, user, userName, user_name, login, account, userid, user_id

Yöntem 4: WordPress Entegrasyonu

<?php
// WordPress tema dosyanızda (functions.php veya template)
$current_user = wp_get_current_user();
$username = $current_user->user_login;
?>

<iframe id="makiIframe" src="http://localhost:3000"></iframe>

<script>
document.getElementById('makiIframe').addEventListener('load', function() {
    this.contentWindow.postMessage({
        type: 'setUsername',
        username: '<?php echo esc_js($username); ?>'
    }, '*');
});
</script>

Yöntem 5: React/Next.js Entegrasyonu

import { useEffect, useRef } from 'react';
import { useUser } from '@/hooks/useUser';

export default function ContactForm() {
    const iframeRef = useRef(null);
    const { user } = useUser(); // Kendi user hook'unuz
    
    useEffect(() => {
        const handleLoad = () => {
            if (iframeRef.current && user?.username) {
                iframeRef.current.contentWindow.postMessage({
                    type: 'setUsername',
                    username: user.username
                }, '*');
            }
        };
        
        const iframe = iframeRef.current;
        iframe?.addEventListener('load', handleLoad);
        
        return () => iframe?.removeEventListener('load', handleLoad);
    }, [user]);
    
    return (
        <iframe 
            ref={iframeRef}
            src="http://localhost:3000"
            width="100%"
            height="700"
            style={{ border: 'none', borderRadius: '8px' }}
        />
    );
}

🔒 Güvenlik - Origin Kontrolü (Önerilen)

Production'da mutlaka origin kontrolü yapın:

<script>
const iframe = document.getElementById('makiIframe');

iframe.addEventListener('load', () => {
    // Sadece güvendiğiniz domain'e mesaj gönderin
    iframe.contentWindow.postMessage({
        type: 'setUsername',
        username: 'john_doe'
    }, 'https://your-domain.com'); // '*' yerine gerçek domain
});
</script>
