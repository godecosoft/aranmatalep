🧪 iFrame Entegrasyon Test Sayfası

Bu sayfa kendi sitenizi simüle eder. Gerçek entegrasyon öncesi burada test edin.

📌 Bu Sayfa Nedir?

  • Kendi sitenizi simüle eder - Gerçek entegrasyon öncesi test yapabilirsiniz
  • Kullanıcı adı gönderme - postMessage ve URL yöntemlerini test edin
  • Canlı test - Değişiklikleri anında görün
  • Güvenli - Sadece lokal bilgisayarınızda çalışır
postMessage (Önerilen)
URL Parametresi

🚀 postMessage Yöntemi EN İYİ

✅ Kendi sitenizde bu kodu kullanın:

<iframe id="makiIframe" src="http://YOUR-DOMAIN.com"></iframe>

<script>
// Kullanıcı adınızı buradan alın (örnek)
const username = localStorage.getItem('username'); 
// veya: const username = '';
// veya: const username = document.querySelector('.user-name').textContent;

const iframe = document.getElementById('makiIframe');

iframe.addEventListener('load', () => {
    // iframe'e kullanıcı adını gönder
    iframe.contentWindow.postMessage({
        type: 'setUsername',
        username: username
    }, '*'); // Production'da '*' yerine gerçek domain kullanın
});
</script>

🔗 URL Parametresi Yöntemi

✅ Kendi sitenizde bu kodu kullanın:

<!-- Statik HTML -->
<iframe src="http://YOUR-DOMAIN.com?username=kullanici_adi"></iframe>

<!-- JavaScript ile -->
<script>
const username = 'john_doe'; // Kendi sisteminizdeki kullanıcı
const iframe = document.createElement('iframe');
iframe.src = `http://YOUR-DOMAIN.com?username=${encodeURIComponent(username)}`;
document.body.appendChild(iframe);
</script>

<!-- PHP ile -->
<iframe src="http://YOUR-DOMAIN.com?username=<?php echo urlencode($username); ?>"></iframe>
